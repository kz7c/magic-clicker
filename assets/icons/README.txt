アイコン差し替え手順（ピクセル風）

重要:
- ACイラスト等の画像は「直リンク」せず、必ずダウンロードしてこのフォルダに保存してください。
- ライセンス・クレジット表記は各サイトの規約に従ってください。

保存先と命名:
- 素材: assets/icons/materials/<material_id>.png
- ポーション: assets/icons/potions/<potion_id>.png
- ボックス: assets/icons/boxes/<box_id>.png
- 装備: assets/icons/equipments/<equipment_id>.png

例:
- assets/icons/materials/magic_herb.png
- assets/icons/potions/mana_boost.png
- assets/icons/boxes/common_box.png
- assets/icons/equipments/eq_wand_apprentice.png

補足:
- ファイルが存在しない場合は、ゲーム内の自動ピクセルアイコンにフォールバックします。
- 推奨サイズ: 32x32 または 64x64（PNG, 背景透過）
